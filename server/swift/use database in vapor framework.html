use database in vapor framework

The official document does not cover the detail of how to use database. After some trial work,

Vapor version: 2.3.0

1 create a model

Create a model as the official document described. Assume it is Pet.

2 set the database

sqlite3 is contained in the FluentProvider, so let us take it as example. In the Config+Setup.swift, add: 

   let driver = try SQLiteDriver()
        let db = Database(driver)
        try Pet.prepare(db)
        Pet.database = db

3 add config files

We need a fluent.json and sqlite.json file in Config folder. 

# sqlite.json
{
  "path": "./Database/vapor.db"
}

#fluent.json
{
  "driver": "sqlite"
}

4 test

Add a new route: 

  builder.get("pet") { req in
            let pet = try Pet.find(0)
            return pet.makeJSON()
        }

Now run vapor build and vapor run server, things should work right.
