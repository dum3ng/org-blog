install nginx and php in osx

1 install nginx

Use homebrew: 

  brew install nginx

1.1 configure nginx

Run command nginx to start nginx service. Edit the configuration file, generally the file is /usr/local/etc/nginx/nginx.conf. The default configuration send content in a html folder, which you can find it in /usr/local/Cellar/nginx/${version_number}/. The simplist configuration would be: 

  worker_processes  1;

  events {
      worker_connections  1024;
  }

  http {
      include mime.types;
      default_type  application/octet-stream;
      sendfile        on;
      log_format  main  '$remote_addr - $remote_user [$time_local] "$request" '
                       '$status $body_bytes_sent "$http_referer" '
                       '"$http_user_agent" "$http_x_forwarded_for"';
      access_log  logs/access.log  main;
      index  index.html index.htm;
      root    /site/php/data;
              
      server {
          listen 8080;
          server_name localhost;

          location / {
             
          }
          location /images {
            try_files $uri $uri/ ;
          }

      }
  }

Yet you can ignore the log, but it would be very useful if something went wrong. The log file can be found at /usr/local/var/log/nginx/.

Run command nginx -s reload to reload the configuration file to see the result. Note: If something works not as expected, you may need restart the nginx service. Just run: 

  nginx -s stop
  nginx

2 install php

Follow the instruction at [github](https://github.com/Homebrew/homebrew-php).

After installed successfully, start the cgi service: 

  php-cgi -b 9000

3 connect nginx with php cgi

Re-edit the nginx.conf file, add a location block: 

    location ~ \.php$ {
       fastcgi_pass   127.0.0.1:9000;
       fastcgi_index  index.php;
       fastcgi_param  SCRIPT_FILENAME  scripts$fastcgi_script_name;
       include        fastcgi_params;
    }

The tidy before the location means this is a regular expression match. For detail, you can see a article by [digital ocean](https://www.digitalocean.com/community/tutorials/understanding-nginx-server-and-location-block-selection-algorithms) Note: Take a look at the fastcgi_param option, make sure it is in your own path. The full file struct now will be like this: 

  /site/php/data
                /images
                /scripts

4 Done!

Now write a simple php file in the scripts folder, open localhost/my.php in your browser and see the result.
